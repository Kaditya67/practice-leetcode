# Algorithmic Mastery & Data Structures Repository

Welcome to the central repository for Data Structures and Algorithms practice. This repository serves as a structured, professional learning management system designed for continuous growth and algorithmic excellence.

---

## Directory Architecture

The repository is strictly divided into two distinct historical and active workspaces to prevent clutter and maintain focus.

### 1. The Active Workspace
**Directory: `/2026/`**

This is the primary zone for current algorithmic training. It has been meticulously structured into a 15-module curriculum, progressing from fundamental C++ Standard Template Library (STL) usage to advanced concepts like Dynamic Programming on Trees and Heavy-Light Decomposition.

Key features of the `/2026/` workspace include:
- **Atomic Study Guides**: Every module directory contains a bespoke `STUDY_GUIDE.md` detailing the required logic, prerequisites, and common pitfalls for that specific topic.
- **Problem Solving Pedagogy**: A codified methodology (`PROBLEM_SOLVING_PEDAGOGY.md`) for approaching, brainstorming, and implementing solutions derived from competitive programming best practices.
- **Cognitive Optimization Tools**: Integrated systems for spaced repetition (`SPACED_REPETITION_LOG.md`) and rational debugging (`DEBUGGING_MANUAL.md`).
- **Comprehensive Master Plan**: A granular checklist (`MASTER_PLAN.md`) that links specific classic problems to their theoretical concepts.

*To begin or continue learning, navigate immediately to the `2026` directory and consult its inner `README.md`.*

### 2. The Historical Archives
**Directory: `/Archives/`**

This directory contains legacy code, previous roadmaps, and early implementations. It has been categorized by data structure (Arrays, Strings, Trees, etc.) for easy reference and retrieval.

While the `/Archives/` section contains valuable historical context and over 150 solved problems, active development and structured learning should take place exclusively within the `/2026/` directory.

---

## Philosophy

The philosophy governing this repository is simple: **Precision over Speed. Methodology over Memorization.**

By adhering to the strict architectural guidelines outlined in the active workspace, the goal is to transform from relying on intuition to relying on a proven, systematic approach to algorithmic problem-solving.

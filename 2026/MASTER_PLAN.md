## Atomic Checklists (The Journey)

### 01. Foundations & STL (Folder 01)
- [ ] Complexity: O(1), O(log N), O(N), O(N log N), O(N^2), O(2^N), O(N!).
- [ ] Vectors: Amortized complexity, capacity vs size.
- [ ] Iterators: begin(), end(), reverse_iterator.
- [ ] Maps/Sets: std::map (Log N) vs std::unordered_map (Avg 1).
- [ ] Custom Priority Queue: Using lambda or struct comparators.

### 02. Mathematics (Folder 02)
- [ ] Sieve of Eratosthenes: Basic (O(N log log N)).
- [ ] Segmented Sieve: For large ranges (L, R).
- [ ] Modular Arithmetic: (A+B)%M, (A-B)%M, (A*B)%M.
- [ ] Modular Inverse: Fermat's Little Theorem.
- [ ] Matrix Exponentiation: Fibonacci in O(log N).
- [ ] Bitwise XOR: Master the property X ^ X = 0.

### 03. Arrays & Searching (Folder 03)
- [ ] Sliding Window (Shrinkable): While condition while (total > K) left++.
- [ ] Sliding Window (Non-Shrinkable): Keep the best window size.
- [ ] Two Pointers (Meet in Middle): 3-Sum, 4-Sum patterns.
- [ ] Binary Search (Classic): Standard while (l <= r).
- [ ] Binary Search (Threshold): Use bool check(val) to find the first/last true.

### 11. Dynamic Programming (Folder 11)
- [ ] 1D DP: Jump Game, House Robber.
- [ ] 2D DP: Longest Common Subsequence, Edit Distance.
- [ ] Knapsack (0/1): Every item used exactly once.
- [ ] Knapsack (Unbounded): Every item used multiple times.
- [ ] DP on Trees: Maximum Path Sum, Independent Set.
- [ ] Digit DP: Find sum of digits of all numbers between L and R.

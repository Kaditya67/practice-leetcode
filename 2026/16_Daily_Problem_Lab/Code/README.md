# Daily Problem Lab Code

Use this folder for random practice solutions from any platform.

## Naming Convention
- `Platform_ProblemID_Title_V1.cpp`
- `Platform_ProblemID_Title_V2.cpp`
- `Platform_ProblemID_Title_Final.cpp`

## Required Header Comment
Every file should start with:

```cpp
/*
 * PLATFORM: [LeetCode / CodeChef / Codeforces / etc.]
 * PROBLEM: [ID + Name]
 * TOPIC: [Main topic]
 * VERSION: [V1 / V2 / Final]
 * CORE IDEA: [One-line logic]
 * COMPLEXITY: [Time / Space]
 */
```

## What To Keep Here
- brute-force version if useful
- improved version
- final accepted version
- optional re-solve version after revision

## What Not To Keep Here
- long theory dumps
- chapter notes

Theory belongs in `../Notes/`.

# 16 Daily Problem Lab

Use this folder for any random or daily problem from any platform:
- LeetCode
- CodeChef
- Codeforces
- GeeksforGeeks
- AtCoder
- HackerRank

This folder is intentionally separate from the ordered chapter folders.
It is for:
- daily challenge problems
- random practice
- mixed-topic revision
- problems you want to study deeply with multiple versions

## Folder Structure
- `Code/`
  - final accepted solutions
  - alternate versions
  - brute-force to optimized progression
- `Notes/`
  - intuition
  - what you learned
  - mistakes made
  - pattern recognition

## Recommended Workflow
1. Create one note file from `Notes/TEMPLATE.md`.
2. Create one code file from the naming convention in `Code/README.md`.
3. After solving, write:
   - first intuition
   - final insight
   - mistakes
   - better version if any
4. If the problem teaches a reusable pattern, link it back to the proper chapter mentally.

## Naming Suggestion
For notes:
- `YYYY-MM-DD_Platform_ProblemName.md`

Examples:
- `2026-03-30_LeetCode_2840_CheckStringsEqual.md`
- `2026-03-30_CodeChef_StarterProblem.md`

For code:
- `Platform_ProblemID_Title_V1.cpp`
- `Platform_ProblemID_Title_V2.cpp`
- `Platform_ProblemID_Title_Final.cpp`

Examples:
- `LeetCode_2840_CheckStringsEqual_V1.cpp`
- `LeetCode_2840_CheckStringsEqual_Final.cpp`

# Code Repository: 15_Contests_and_Assessments

## Organization Logic
Save problems using the format: `AtomicTopic_ProblemID_Title.cpp`
Example: `BinarySearch_LC1011_ShipCapacity.cpp`

## Required Header Comment
Every file MUST start with:
`cpp
/*
 * TOPIC: [Specific Atomic Topic]
 * CORE TRICK: [The 1-sentence logic that solves the problem]
 * COMPLEXITY: [Time/Space]
 */
`

## 🎯 Goal
Clean, reusable, and well-commented solutions. No messy code here.
